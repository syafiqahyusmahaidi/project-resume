Web Scraping

EXPLANATION

Web scraping is the process of extracting data from a website. This data is gathered and then
exported into a format that is more user-friendly. It doesn't matter if it's a spreadsheet or an API.
Although web scraping can be done manually, some websites have too many data to extract and we
want to save time by just doing it automatically using web scraping tools. For some cases, web
scraping is not a simple task. Some websites do have complicated interface which in many shapes
and forms. It will take too much time if we do it manually but we can save time by using web
scraping tools because it already has variety of functionality and features.

For example, we can use web scraping to export a list of products names and prices from Shopee
onto an Excel spreadsheet.

How Web Scraper Works?
  • Web scraper will get one or more URLs to load before scraping.
  • After that, the scraper loads the whole HTML code for the given page. The full website,
    including CSS and Javascript parts, will be rendered by more advanced scrapers.
  • The scraper can extract all the data or can only extract the data that the user want before
    the project is run
OCTOPARSE

Octoparse is a powerful data extraction tool without any coding needed. We can easily find anything
that we want to extract in the google and then choose template in the Octoparse and also choose
where we want to extract it.

Steps :

  • Choose the template in the Octoparse that we already download in our computer.
  • Read the instruction and choose which element that we want to extract from web as
    example the title only, or the discription only
  • Start extraction. Choose local extraction if it is a one-time project.
  • Choose format of the extraction such as Excel 2007, HTML, or JSON.
